# Investment Future Value Calculator

A desktop investment calculator built with Python that estimates the future value of an investment using compound interest and monthly contributions.

## Features

- Calculate future portfolio value
- Monthly investment (SIP) support
- Compound interest calculations
- Interactive portfolio growth chart
- Dark and light mode
- Indian Rupee (₹) formatting
- Clear/Reset functionality

## Requirements

- Python 3.x
- customtkinter
- matplotlib

Install the required libraries:

```bash
pip install customtkinter matplotlib
```

## Run

```bash
python investment_calculator.py
```
